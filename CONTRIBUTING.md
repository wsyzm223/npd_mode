# 参与贡献

欢迎对「隐性 NPD 母亲内在程序模型」提出想法、修正或补充。

## 如何参与

- **提建议 / 报错**：在 [Issues](issues) 里新建一个 Issue，描述你的想法或遇到的问题即可。
- **改代码 / 文案**：
  1. **Fork** 本仓库到你的账号下；
  2. 在你的 Fork 里创建分支、修改代码；
  3. 向本仓库提交 **Pull Request**。

我会对每个 PR 进行查看，通过后才会合并到主仓库。若有需要修改的地方，会在 PR 里留言说明。

## 注意事项

- 本项目为疗愈向、教育向，请勿提交模拟具体伤害性对话或可能对他人造成二次伤害的内容。
- 如需大改结构或新增模块，建议先在 Issues 里讨论一下方向，再动手写代码。

感谢你的参与。
